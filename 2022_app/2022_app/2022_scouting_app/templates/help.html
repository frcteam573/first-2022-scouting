How-to

Create Excel file from database

Make sure you have Python installed!

Check if Python is present by opening up Comandline Promt and running:

python --version

If Python is present, it should return something like:

python --version
Python 3.10.2

Run the following command in Commandline Prompt:

cd 2022_app\2022_app\2022_scouting_app\leaderscripts
python .\CreateExcelFile.py

A ScoutingData_DATE.xlsx will be generated in the same directory.
